Embed your Resume/CV on your blog

== Description ==

This plugin allows you to embed your ResumePark hosted resume on your blog.  You must be a ResumePark user to use this plugin.

== Installation ==

1. Upload the resumepark folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Screenshots ==

1. ResumePark Resume

== Usage ==

Add the resumpark tag to any post / page in which you'd like to embed your resume

The tag takes the following 3 paramaters:

1.  resumeId - retrieved from the ResumePark share page
3.  height (pixels)
4.  width (pixels)

Sample Tag: [resumepark resumeId=kcKXjGhQzE6XHsVZ-vTM6Q_41 height=500 width=690 /]
